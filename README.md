# PWM-Dead-Time-Calculator
Contains PWM Dead Time Calculator with Nomenclature according to 
<a href="https://www.researchgate.net/publication/400067681_PWM_Dead_Times_in_Automotive_Traction_Inverters_using_IGBT_SiC_MOSFET_or_SiSiC_Fusion_Switch_Power_Modules" target="_blank">T. Reiter, J. Schapdick, M. Krug, M. Muenzer, and F. Wolter, “ PWM Dead Times in Automotive Traction Inverters using IGBT, SiC MOSFET, or Si/SiC Fusion Switch Power Modules.” IET Power Electronics 19, no. 1 (2026): e70185. https://doi.org/10.1049/pel2.70185</a>  <br>

# How to use
* Start MATLAB (tested version R2022b)
* Install the *.mlappinstall
* Start the App
